# ETX输入工具 - Vue+Electron版本

这是ETX远程输入工具的Vue+Electron版本，提供了跨平台支持和现代化的用户界面。

## 功能特点

1. 全局快捷键监听（默认F2）
2. 置顶的输入窗口，支持透明效果
3. 文本输入和发送到ETX环境
4. 系统托盘图标和菜单
5. 可自定义快捷键设置
6. 开机启动设置
7. 跨平台支持（Windows/macOS/Linux）

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 运行应用：
```bash
npm start
```

3. 开发模式运行：
```bash
npm run dev
```

## 使用方法

1. 启动应用后，程序会在系统托盘中显示图标
2. 按下F2键（或自定义快捷键）显示输入窗口
3. 在输入框中输入要发送到ETX的内容
4. 按Enter键发送，文本将复制到剪贴板
5. 在ETX环境中按Ctrl+V粘贴内容
6. 按Esc键或点击取消按钮关闭输入窗口

## 系统托盘操作

右键点击系统托盘图标可以：
- 显示输入窗口
- 打开设置界面
- 切换开机启动
- 退出应用

## 设置选项

在设置界面中可以：
- 自定义全局快捷键
- 设置开机启动

## 注意事项

1. 由于安全限制，应用无法直接模拟键盘输入到其他程序
2. 文本会复制到剪贴板，需要手动粘贴到ETX环境中
3. 应用会自动保存和恢复之前的剪贴板内容
4. 首次运行时可能需要授权系统权限

## 技术架构

- 前端：Vue 3 + HTML + CSS
- 后端：Electron
- 跨平台：支持Windows、macOS和Linux
- 配置存储：用户目录下的.json文件
